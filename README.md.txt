# Marketing Funnel & Conversion Performance Analysis

## Project Overview

This project analyzes customer conversion behavior using the Bank Marketing Dataset and Power BI.

The objective is to identify conversion trends, customer segments, and opportunities to improve marketing performance.

## Tools Used

* Power BI
* DAX
* Excel/CSV

## Key KPIs

* Total Contacts
* Converted Customers
* Non-Converted Customers
* Conversion Rate

## Dashboard Features

* Marketing Conversion Funnel
* Contact Method Analysis
* Job-wise Conversion Analysis
* Monthly Conversion Trends
* Education-wise Analysis
* Housing Loan Impact Analysis

## Key Insights

* Majority of customers did not subscribe.
* Overall conversion rate was approximately 11.7%.
* Cellular contact methods generated the highest conversions.
* Some professions showed significantly higher conversion rates.
* Customers without loans converted more frequently.

## Recommendations

* Focus campaigns on high-converting customer segments.
* Increase outreach through effective contact methods.
* Improve targeting during high-performing months.
* Develop personalized marketing strategies.
